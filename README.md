## Connecd - server

Server for [Connectd](https://github.com/HanMoeHtet/connectd) app.

## Get started

### Clone the repo

```
 git clone https://github.com/HanMoeHtetconnectd-server.git
```

### Install dependencies

```
 npm install
```

### Copy env file and fill in the values

```
  cp .env.example .env
```

###  Run the app

```
 npm run dev
```
